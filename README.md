## 🎯 Overview

This is a production-ready, fully responsive portfolio website designed for modern developers working at top tech companies like Google, Microsoft, and Apple. The portfolio showcases a clean, minimal UI with a dark theme, gradient highlights, glassmorphism effects, and smooth animations.

### Key Highlights

✅ **100% Vanilla JavaScript** - No frameworks or dependencies  
✅ **Fully Responsive** - Mobile, tablet, and desktop optimized  
✅ **Dark Modern Theme** - Silicon Valley aesthetic  
✅ **Glassmorphism UI** - Modern card designs with blur effects  
✅ **Smooth Animations** - Scroll reveals, typing effects, and transitions  
✅ **Live GitHub Integration** - Fetches real repositories via GitHub API  
✅ **Performance Optimized** - Fast loading, lazy loading, and debounced events  
✅ **SEO Ready** - Semantic HTML and meta tags  
✅ **Accessible** - ARIA labels and keyboard navigation support

## 📦 Installation

### Quick Start

1. **Clone or Download** the repository
2. **Open `index.html`** in your browser
3. That's it! No build process required.

### File Structure

```
portfolio/
├── index.html          # Main HTML file
├── css/
│   └── style.css      # All styles (1200+ lines)
├── js/
│   └── script.js      # All functionality (700+ lines)
├── assets/
│   └── images/        # (Optional) Add your images here
└── README.md          # Documentation
```

### Local Development

Simply open `index.html` in any modern browser:

```bash
# Option 1: Direct file open
open index.html

# Option 2: Use a local server (recommended)
python -m http.server 8000
# Then visit: http://localhost:8000

# Option 3: VS Code Live Server
# Install "Live Server" extension and click "Go Live"
```

## 🎨 Customization

### Personal Information

**Update these values in `index.html`:**

```html
<!-- Line 78: Hero name -->
<h1 class="hero-title">
    Hi, I'm <span class="gradient-text">Your Name</span>
</h1>

<!-- Line 89: Contact email -->
<a href="mailto:your.email@example.com" aria-label="Email">
```

### GitHub Integration

**Update username in `js/script.js`:**

```javascript
// Line 23
const config = {
    githubUsername: 'your-github-username', // Change this
    githubRepoLimit: 6
};
```

### Color Scheme

**Update colors in `css/style.css`:**

```css
/* Line 12 - CSS Variables */
:root {
    --color-accent: #667eea;        /* Primary accent color */
    --color-accent-2: #f093fb;      /* Secondary accent */
    --color-accent-3: #4facfe;      /* Tertiary accent */
    --gradient-primary: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}
```

### Projects

Update project information in `index.html` (lines 443-576):

```html
<div class="project-card" data-color="purple">
    <div class="project-number">01</div>
    <div class="project-banner purple-gradient">
        <!-- Update icons -->
    </div>
    <div class="project-content">
        <h3 class="project-title">Your Project Name</h3>
        <p class="project-description">Your project description</p>
        <div class="project-tech">
            <span class="tech-tag">Tech 1</span>
            <!-- Add more tech tags -->
        </div>
        <div class="project-links">
            <a href="your-github-link" class="project-link">GitHub</a>
            <a href="your-demo-link" class="project-link primary">Live Demo</a>
        </div>
    </div>
</div>
```

### Profile Image

Replace the profile placeholder in `index.html` (line 188):

```html
<!-- Current: Icon placeholder -->
<div class="profile-placeholder">
    <i class="fas fa-user-circle"></i>
</div>

<!-- Replace with: Your image -->
<div class="profile-placeholder">
    <img src="assets/images/profile.jpg" alt="Your Name">
</div>
```

Add this CSS to maintain circular shape:

```css
.profile-placeholder img {
    width: 100%;
    height: 100%;
    object-fit: cover;
    border-radius: 50%;
}
```

### Option 1: GitHub Pages (Free)

2. Go to Settings → Pages
3. Select branch and root folder
